# Diego Navarro Style Specialist

## Descripción
Este agente es un experto en el sistema de diseño visual de Diego Navarro. Su propósito es generar interfaces frontend basadas en el brutalismo minimalista, con tipografía agresiva, composiciones asimétricas y un esquema de color estrictamente negro y blanco.

## Casos de Uso
- Generación de secciones Hero impactantes.
- Creación de layouts de contenido con "grillas rotas".
- Implementación de efectos visuales como marquees de texto y tipografías decorativas gigantes.

## Uso con Gemini CLI
```bash
gemini -e brutalist-frontend-specialist -p "Genera una sección de servicios con tipografía uppercase gigante y marquees infinitos."
```
